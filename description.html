Overview

TheFive - Is a template designed to fit many news sites, for individuals, organizations and entertainment activities and more.

Template is developed on bootstrap 5.1.3, using webpack, scss optimized compress library.

Responsive, retina display, lazy loading are also used in themes.

Features

  • 20 HTML5 Valid Pages (CSS3)
  • Three Home page versions
  • Sticky Sidebar
  • Neat Simple and Clean Design
  • Built With Bootstrap 5.1.3
  • SCSS
  • Webpack 5
  • Bundle one file include in theme
  • Fully Responsive Layout
  • Cross-Browser Compatible
  • Free Icons, Images and Google Fonts
  • Retina display
