HTML Structure

The Five follows a simple and easy to customize coding structure. Here is the sample for your reference:
The template is based on Bootstrap Framework

                
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
    <title>The Five - Blog & Magazine Minimal</title>

    <link href="/" rel="canonical">
    <link href="assets/images/favicon.png" rel="shortcut icon" type="image/vnd.microsoft.icon">

    <!--- start style --->
    <link href="assets/build/css/main.min.css" media="screen" rel="stylesheet" type="text/css">
    <!--- end style --->

</head>
<body>


    <!-- start loading-->
    <div class="loading"></div>
    <!-- end loading-->

    <!--Start Block Head-->
    <header class="head"></header>
    <!--End Block Head-->

    <!-- Start block area feature-->
    <section class="area-feature"></section>
    <!-- End block area feature-->

    <!-- Start block area post nine-->
    <section class="area-post-nine"></section>
    <!-- End block area post nine-->

    <!-- Start block area post sidebar-->
    <section class="area-post-sidebar"></section>
    <!-- End block area post sidebar-->

    <!-- Start block area post six-->
    <section class="area-post-six"></section>
    <!-- End block area post six-->

    <!-- Start block area post sidebar two-->
    <section class="area-post-sidebar-two"></section>
    <!-- End block area post sidebar two-->

    <!-- Start footer-->
    <section class="footer"></section>
    <!-- End footer-->

    <!--- start script bundle --->
    <script src="assets/build/js/main.bundle.js"></script>
    <!--- end script bundle --->

</body>
</html>

If you need more information, please visit bootstrap site: https://getbootstrap.com

Development

Prerequisites

Note you have to install node and npm
When we develop this project it is used for version: Node 14.17.6 & NPM 6.14.15

1. In the project directory HTML, you can run:

npm install

2. Launches the test runner in the interactive watch mode.

npm run watch

3. Builds the app for production to the build folder.

npm run build
